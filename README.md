# 小新商城

> 小新商城是一套基于uniapp+uniCloud开发的单商户商城系统，通过拖拽可视化组件，生成适用全行业多场景模板，帮商家快速网上开店。同时支持PC端后台管理和移动端后台管理，提高产品全方位用户体验。

## 支持组件
* 基础组件：文本、图片广告、商品、视频、标题、图文导航、商品搜索、公告、辅助空白、辅助线、自由面板
* 系统组件：分类模板、页面导航
* 营销组件：优惠券
